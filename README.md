# ZOJ Tourist

about：
1. 为ZOJ的题目提供一份答案托管
2. 旨在让各位愉快且较有效率的刷题

## 代码结构
- src(存放各位已经ac的代码)
	- $(zoj-question-id)
		- README.md(用于题目描述)
		- $(your-username)
			- $(any-files-about-your-answers)
- utils(相关任务脚本)
